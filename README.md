**Meme Generator** <br/>

<Summary>The Meme Generator is a web application built with React.js that allows users to create custom memes through a simple form. Users can input an image URL to set the background, add top and bottom text, and adjust the font size for customization. After clicking the submit button, the application dynamically displays a new UI with the generated meme using the provided inputs. Featuring a user-friendly interface and instant updates, this project is ideal for practicing React basics while having fun creating memes.
</summary>
